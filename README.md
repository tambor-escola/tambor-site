# Tambor — Artes Visuais

Welcome to the repository for the Tambor website.

Tambor is a space dedicated to art, creativity and artistic practice in Lisbon. The website provides information about courses, workshops, events and upcoming activities.

## Technology

* The website is built with [Jekyll](https://jekyllrb.com/).
* It uses custom styling.
* Jekyll requires [Ruby](https://www.ruby-lang.org/).
* Ruby dependencies are managed with [Bundler](https://bundler.io/).
* The site is hosted on GitHub Pages.
